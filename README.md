# AkelPad
Most probably the best Notepad ever created !

This version is ONLY for x64 !
It has been modified to be built on Visual Studio 2022 Community !

It includes:
Redirect (ability to Redirect from Notepad.exe to AkelPad.exe)

And Plugins:
Coder
Exit
Fonts
Format
FullScreen
Hotkeys
LineBoard
Scripts
Scroll
SmartSel
SpecialChar
Speech
Stats
ToolBar
